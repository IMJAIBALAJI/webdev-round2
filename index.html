Tech Fest 2025: Web Development Challenge

Round 2 - Hard Level Questions

Instructions

Congratulations on qualifying for Round 2! Solve the 2 questions below to showcase your advanced web development skills. Submit your solutions as working code (HTML, CSS, JavaScript) via the provided submission link. Time limit: 4 hours. Use standard web technologies and public APIs as specified. Ensure your solutions are functional and responsive. Best of luck!

Question 1: Real-Time Weather Dashboard

Build a weather dashboard using a public API (e.g., OpenWeatherMap). The dashboard should:

  • Allow users to input a city name to display current weather (temperature, humidity, description).
  • Show a 5-day forecast with date, temperature, and weather icon.
  • Save recent searches in a dropdown menu, stored in localStorage for persistence.
  • Handle errors (e.g., invalid city) with user-friendly messages.
  • Be styled to be visually appealing and responsive across devices.

Output: A functional weather dashboard with all features.

Question 2: Interactive Data Table with Filtering

Create a data table displaying 50 users (use mock data or a public API like JSONPlaceholder). Each row should show name, email, and phone number. Implement:

  • A search bar to filter users by name, updating in real-time as the user types.
  • Sort functionality for each column (ascending and descending).
  • Pagination to show 10 users per page.
  • A responsive and clean table design.

Output: A working table with filtering, sorting, and pagination.
