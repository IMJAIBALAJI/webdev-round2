Instructions

  • Duration: 1.3 hours.

  • Use only HTML, CSS, and JavaScript (no frameworks or APIs).
  • Create one HTML file named round2.html.
  • Test locally or on GitHub Pages (https://username.github.io/repo).
  • Submit file via [submission link] by [deadline].
  • Judging: Functionality (50%), Creativity (30%), Design (20%).

Question 1: Chat App Interface

Create a chat app interface with three pages: Login, Chat, and Contact. The Login page should have a form with username and password fields, validated on submit (username > 3 chars, password > 5 chars) using JavaScript. The Chat page should display a message area (static messages) and an input to add new messages (append locally). The Contact page should list 3 contacts with names and emails. Use a navigation bar to switch pages (single HTML file). Style with fest colors, smooth transitions, and hover effects (e.g., button glow).

Hint: Use display for page switching, appendChild for messages.

Question 2: Movie Ticket Booking Interface

Build a movie ticket booking interface with three pages: Movie Selection, Seat Booking, and Contact. The Movie Selection page should list 3 movies with titles and times (click to select). The Seat Booking page should show a 3x3 grid of seats (click to toggle select/deselect, max 3 seats) with a counter. The Contact page should have a form for name and email, validated on submit (non-empty). Use a navigation bar to switch pages (single HTML file). Style with fest colors, animations (e.g., seat flip), and hover effects.

Hint: Use classList.toggle for seats, querySelector for validation.

Question 3: Event Registration Portal

Design an event registration portal with three pages: Sign-Up, Event List, and Support. The Sign-Up page should have a form with name, email, and event dropdown (3 options), validated on submit (email format, non-empty). The Event List page should display 3 events with titles, dates, and a “Join” button (alerts “Joined!” on click). The Support page should show a FAQ with 3 questions (click to toggle answers). Use a navigation bar to switch pages (single HTML file). Style with fest colors, slide-in animations, and hover effects.

Hint: Use regex for email, onclick for toggles.
